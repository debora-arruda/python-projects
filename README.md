# 🎮 Python Training — Jogos & Lógica

Repositório de projetos em Python focados em lógica de programação, estruturas condicionais e interface gráfica. Desenvolvido como parte dos meus estudos em Análise e Desenvolvimento de Sistemas.

---

## 📂 Projetos

### ✊ Pedra, Papel e Tesoura

Jogo clássico implementado em duas versões:

- **Terminal** (`rock_paper_scissors.py`) — versão via linha de comando com entrada do usuário e lógica de vitória/derrota contra o computador
- **Interface Gráfica** (`rock_paper_scissors_gui.py`) — mesma lógica, agora com interface visual usando Tkinter

**Conceitos aplicados:** estruturas condicionais, manipulação de entrada do usuário, módulo `random`, criação de GUI com Tkinter

### 🔤 Initials

Programa que trabalha com manipulação de strings em Python.

---

## 🛠️ Tecnologias

- Python 3
- Tkinter (GUI)
- Módulo `random`

---

## ▶️ Como executar

```bash
# Clone o repositório
git clone https://github.com/debora-arruda/Python-Training.git

# Versão terminal
python rock_paper_scissors.py

# Versão com interface gráfica
python rock_paper_scissors_gui.py
```

---

## 📚 O que aprendi

- Lógica condicional (`if/elif/else`) para definir regras do jogo
- Geração de escolhas aleatórias com `random.choice()` 
- Criação de interfaces gráficas simples com Tkinter
- Manipulação de strings e entrada do usuário
